# Installation and requirements

Install the Adaptive Performance package and review its compatibility with the Unity Editor.

## Install the package

You can install this package using the Package Manager. Follow the instructions to [install a UPM package from a registry](xref:upm-ui-install) in the Unity Editor.

## Unity Editor compatibility

Adaptive Performance is a [core Unity package](https://docs.unity3d.com/Manual/pack-core.html). This means its version is tied to the Unity Editor version, and the Editor automatically provides the correct version of the package.
